# CSS Color Markers

A visually engaging web project demonstrating the power of **CSS3 gradients**, **flexbox**, and **box-shadows** to create realistic, three-dimensional markers using pure code.

## Features
* **Pure CSS Art:** No external image files used for the markers.
* **Linear Gradients:** Realistic lighting and depth effects.
* **Responsive Design:** Styled to look great on any screen size.
* **Shadow Effects:** Soft glow and drop shadows for a "floating" feel.

## How to Use
1. Clone the repository.
2. Open `index.html` in your favorite web browser.
3. Customize the colors in the CSS file to create your own marker set!
